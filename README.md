### Справочная информация
`git remote -v` Посмотреть ссылки удаленного репозитория (origin)<br>
`git status` Показывает изменения и их статус<br>
`git log --oneline` Показывает коммиты (Аналог сейво из игр)<br>

#### Подготовить
`git add .` Подготовить к сохранению<br>
`git commit -m ""` Сохранить<br>
`git push -u origin main` Отрпавить коммиты (сейвы) на удаленны репозиторий<br>

### Работа с приложением
`npm run` Показывает доступные команды<br>
`npm ci` (или `npm i`) Устанавливает необходимые пакеты<br>
`npm run start` Запустить проект<br>
Запущенный проект будет доступен по адресу localhost:3000
