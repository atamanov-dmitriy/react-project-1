### Справочная информация
`git remote -v` Посмотреть ссылки на удаленный репозиторий (origin)<br>
`git status` Показывает изменения и их статус<br>
`git log --oneline` Показывает коммиты (Аналог сейвов из игр)<br>

### Подготовить сохранение
`git add .` Подготовить к сохранению<br>
`git commit -m "<Имя коммита>"` Сохранить<br>
`git commit -m "my-first-commit"` Пример команды выше<br>

### Работа с удаленными репозиториями
`git push -u <имя-удаленного-репозитория> <название-ветки-удаленного-репозитория>` Отрпавить коммиты (сейвы) на удаленны репозиторий<br>
`git push -u origin main` Пример команды выше<br>
<br>
`git pull` Получить коммиты (сейвы) из удаленного репозитория<br>
<br>
`git remote set-url <имя-репозитория> <url-ссылка на репозиторий>` Добавить ссылку на удаленный репозиторий<br>
`git remote set-url origin https://github.com/atamanov-dmitriy/react-project-1.git` Пример команды выше<br>
<br>
`git clone <url-ссылка на репозиторий>` Склонировать удаленный репозиторий на локальный компьютер<br>
`git clone https://github.com/atamanov-dmitriy/react-project-1.git` Пример команды выше<br>


### Работа с приложением
`npm run` Показывает доступные команды<br>
`npm ci` (или `npm i`) Устанавливает необходимые пакеты<br>
`npm run start` Запустить проект<br>
Запущенный проект будет доступен по адресу localhost:3000
